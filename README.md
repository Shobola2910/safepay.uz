## ⚡ Quick Deploy

### Option 1 — Vercel CLI
```bash
npm i -g vercel
cd safepay-uz
vercel --prod
```

### Option 2 — GitHub + Vercel
1. Push to GitHub
2. Go to vercel.com → New Project
3. Import your repo
4. Deploy (zero config needed)

### Option 3 — Drag & Drop
1. Go to vercel.com
2. Drag the `safepay-uz` folder
3. Done ✅

## 📲 Demo Flow (for presentations)

1. **Splash** → auto-transitions to onboarding
2. **Onboarding** → 3 slides → Get Started
3. **Auth** → "Demo sifatida kirish" (instant login)

6. **QR Pay** → scan frame → merchant → confirm → ✅
7. **Analytics** → charts, categories, AI insights
8. **Security** → status, toggles, emergency freeze
9. **Wearables** → Ring + Watch status
10. **Rewards** → cashback balance, offers
11. **History** → filters, search, receipts
12. **Profile** → settings, language, logout
